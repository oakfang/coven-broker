# coven-broker
A broker CLI for coven p2p meshes

```
Usage: coven-broker [--json/--silent] [--port=1234]

Options:
  --version   Show version number                                      [boolean]
  --json      Print JSON encoded log messages         [boolean] [default: false]
  --silent    Don't print any logs                    [boolean] [default: false]
  --port, -p  Port to listen on                                  [default: 4000]
  -h, --help  Show help                                                [boolean]                                            [boolean] [default: false]
```
