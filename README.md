# coven-broker
A broker CLI for coven p2p meshes

```
Usage: coven-broker [--json] [--port=1234]

Options:
  --version   Show version number                                      [boolean]
  --port, -p  Port to listen on                                  [default: 4000]
  -h, --help  Show help                                                [boolean]
  --json                                              [boolean] [default: false]
```
